Electronic Flight Instrument System (EFIS)

(January-March 2018, Czech Technical University, Prague, Czech Republic)
My role was the development of the IMU interface of the EFIS(Electronic Flight Instrument System) of UL-39 Aircraft.
Hardware: FMT1030 (ARM M4 based IMU microcontroller)
Software/Programming: C++ and OpenGL

Communication

Development of the low level communication between the computer and the FMT,which is made through XBus Protocol, which is a common binary communication protocol. It is based on messages, which enable to configure the modules and retrieve the output data. 

Data 

Implementation the Real Time Operating System (RTOS) and task scheduling of the system to collect data of orientation, acceleration, angular velocity, magneticism and temperature. For this it was project a state machine.  

User Interface

Communication between the IMU(Inertial Measurenment Unit) board and the User Interface using OpenGL library to plot the graphics.

Tracking and Prediction Algorithm for Surgical Needles

(2017-2018,Laboratory of Robotics and Automation,University of Brasilia, Brasilia, Brazil)
This project is on the development of an algorithm to predict the localization and trajectory of a surgical needle, using a optical tracking sensor to obtain the position of the needle right before the insection and identifying the prediction on the ultrasound image.
Hardware: NDI Polaris Spectra, Ultrasound Transducer
Programming Language: MatLAB

Project

Dual quaternions were used as representation because it is easier to perform operation of translation and rotation.

Development

The algorithm by having the position estimate the position on the ultrasound image, using 3D-2D transformation and camera transformation.

Tests

The algorithm could identify with precison all the positions of the surgical needle and ultrasound transducer.

Green Embassy - Italian Embassy

(2015-2016,Laboratory of Renewable Energy Sources,University of Brasilia, Brasilia, Brazil)
The Italian Embassy in Brazil has a program enabling sustainability for energy and water resources. The solar system of the Embassy is the bigger in Brasilia and has 9 groups of panels and can generate up to 17% of the whole energy consumed of the Embassy. My project was to develop a program to analyse the efficiency of the system and identify failures.
Programming Language: MatLAB

Data Acquiring

Collection of generated data from the inversors, consumed data from the energy provider and metereological data

Data Analysis

Development of a program to analyse of the efficiency by month, analysing more than 4 years of data

Results

It was identified a failure in one of the inversors,by changing the inversor 625 kWh mre by month, saving aproximately 3450 BRL by year (862 euros)

Solar Tracking System

(2016,University of Brasilia, Brasilia, Brazil)                                                            
Solar Tracker is a system built with the goal to move the solar panel to the point of higher solar incidence, enabling higher power generation. To rotate the motor to the right direction four LDR sensors were used, two in each extremity (one turned upways and other one to the side), if the biggest lecture is in one of sensor turned to side, the panel will turn to this direction. If the maximum it is one of the sensor turned up, the difference will be calculated and the panel will turn proportionally to this side.
Hardware: MSP430 MCU(RISC 16 bit), Step Motor, Motor Driver ULN, Light Dependant Resistor (LDR)
Programming Language: C

How Can We Be Recognize?

Pattern Recognition of the Iris

(2017,University of Brasilia, Brasilia, Brazil)                                                            
This project aims to develop a biometry system to recognition of the user through the iris pattern. Computer vision algorithms on segmetation, normalization and codification of images were used.  
Programming Language: MatLaB

Segmentation

A Canny Border Detection was applied.For the segmentation of the iris, a Hough Transform was used to detect the circles around the iris

Codification

The image was normalized and applied Wavelet Gabor Transform with to obtaining only the components of phase 0º of the image , codificating the iris.

Recognition

The Hamming Distance (bit distance) between the images is calculated, if the distance is less than 0.3 the user is considered the same.
